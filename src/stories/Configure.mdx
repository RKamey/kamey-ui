    # Kamey Components

    Esta es la documentación de la librería de kamey-components, la cual contiene componentes de React para ser utilizados en aplicaciones web.

    ## Instalación

    Para instalar la librería de kamey-components, se debe ejecutar el siguiente comando:

    ```bash

    npm install kamey-components

    ```

    ## Uso

    Para utilizar los componentes de kamey-components, se debe importar el componente deseado y utilizarlo en el código de la siguiente manera:

    ```jsx

    import { Button } from 'kamey-components';

    const App = () => {
      return (
        <Button>Click me!</Button>
      );
    }
    
    ```
